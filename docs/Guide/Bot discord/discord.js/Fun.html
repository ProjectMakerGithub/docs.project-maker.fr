Lancer un dé​

    if (command === 'roll') {
        const sides = parseInt(args[0]) || 6;
        if (sides <= 0) {
            await message.channel.send("Le dé doit avoir au moins 1 côté.");
            return;
        }

        const result = Math.floor(Math.random() * sides) + 1;
        await message.channel.send(`Vous avez lancé un dé à ${sides} côtés et obtenu : ${result}`);
    }

Explication : Cette commande permet de simuler le lancer d'un dé virtuel. Vous pouvez utiliser la commande !roll pour lancer un dé à 6 côtés par défaut, ou !roll [nombre de côtés] pour spécifier le nombre de côtés du dé. Le bot renverra le résultat du lancer.

8ball​


    if (command === '8ball') {
        const question = args.join(' ');
        const responses = [
            "C'est certain.",
            "C'est décidément ainsi.",
            "Sans aucun doute.",
            "Oui, absolument.",
            "Pas si sûr.",
            "Impossible à prédire.",
            "Concentrez-vous et demandez à nouveau.",
            "Ne comptez pas là-dessus.",
            "Ma réponse est non.",
            "Mes sources disent non."
        ];
        const response = responses[Math.floor(Math.random() * responses.length)];
        await message.channel.send(`Question : ${question}\nRéponse : ${response}`);
    }

Explication : Cette commande simule une boule magique 8 qui donne des réponses aléatoires aux questions. Vous pouvez utiliser la commande !8ball [votre question] pour poser une question à la boule magique, et le bot renverra une réponse aléatoire.

Ces commandes ajoutent une touche amusante et interactive à votre serveur, permettant aux membres de jouer et de s'amuser avec votre bot. Assurez-vous de remplacer 'YOUR_TOKEN' par le token de votre propre bot.
