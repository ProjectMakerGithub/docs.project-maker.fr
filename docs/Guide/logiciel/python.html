Python

Prérequis Logiciel​

  • Python : 3.10 (L'installation est disponible ici). Si vous n'avez pas Python installé, veuillez suivre ces étapes :

Installation de Python​

astuce

Si vous ne faites pas confiance au lien, vous trouverez toutes les versions ici (dans la section "Files").

  • Windows 10/11
  • Linux
  • MacOS

Windows :

  • Téléchargez la dernière version de Python depuis le site officiel.
  • Lancez l'installateur et assurez-vous de cocher la case "Ajouter Python 3.x au PATH" pendant l'installation.

Linux :

  • Ouvrez le terminal et exécutez les commandes suivantes :

  sudo apt update
  sudo apt install python3.10

MacOS :

  • Téléchargez la dernière version de Python depuis le site officiel.
  • Lancez l'installateur et assurez-vous de cocher la case "Ajouter Python 3.x au PATH" pendant l'installation.

Tester l'installation​

  • Windows 10/11
  • Linux
  • MacOS
  >>> py -V
  3.10
  >>> python3 -V
  3.10
  >>> python3 -V
  3.10
