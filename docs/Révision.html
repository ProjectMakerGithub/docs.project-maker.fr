Commandes Terminal Linux​

ls - Afficher le contenu du dossier  ​

$ ls

cd - Changer de répertoire​

$ cd /    # Accéder à la racine
$ cd ../  # Revenir au dossier parent

mkdir - Créer un nouveau répertoire​

$ mkdir nouveau_dossier

cat - Afficher le contenu d'un fichier​

$ cat fichier.txt

pwd - Afficher le chemin du répertoire actuel​

$ pwd

chmod - Modifier les permissions d'un fichier​

$ chmod +x script.sh  # Ajouter l'exécution au fichier script.sh

chown - Changer le propriétaire d'un fichier​

$ chown nouveau_proprio fichier.txt

chgrp - Changer le groupe d'un fichier​

$ chgrp nouveau_groupe fichier.txt
ownergroupother
rwxrwxrwx
640

ps - Afficher les processus en cours d'exécution​

$ ps aux

kill - Terminer un processus​

$ kill PID

Assurez-vous de remplacer "PID" par le numéro de processus réel.

Vocabulaire :​

  • PID : Processus Identifier
  • PPID : Parent Processus Identifier

Processus :​

  • 4 états : running, waiting, blocked, dead

  • Passage :     - Préemptif : OS décide que running -> waiting     - Non préemptif : OS attend que le processus se bloque

  • Choix de l'élu (processus à exécuter) : suivant l'arrivée ou le temps passé

Exemple de cycle​

ProcessusP1P2P3P4
Instant d'arrivée0237
Durée (en quantum)8622

Règle : un Processus ne peut être exécuté avant son instant d'arrivée
